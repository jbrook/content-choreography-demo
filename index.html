(main)

Using flexbox and box-ordinal-group CSS properties, we can change the order of elements on a page at different viewport sizes. The numbered labels represent the order in which the content appears in the code - resize your browser to see them change if you are on a desktop.

This example is built on the principles of mobile first and The Goldilocks Approach. There are two break-points, one at approximately 33em and another when there is room to display multiple columns. The first breakpoint changes back to the old CSS box model (display:block).

CSS Flexbox (2009 spec) is not very widely supported by browsers but, conveniently, is widely supported by the most used mobile and tablet browsers. It's on those devices where we might want to choreograph the content.

4

(aside)

This example has been built with SASS, Compass and the 'Susy' responsive grid plugin. The mark-up remains semantic and the grid is 'em' based.

Finally, some attention has been paid to typography, golden ratios, vertical rhythm, and modular scale.
